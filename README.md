# Simulados das Aulas do Curso Tecnico

Repositorio com simulados e materiais interativos usados nas minhas aulas do curso tecnico na area de Informatica.

O objetivo deste projeto e organizar atividades de revisao, treino e fixacao de conteudo para apoiar os alunos antes de provas, trabalhos e avaliacoes.

## Objetivo

Este projeto foi criado para:

- disponibilizar simulados por disciplina;
- facilitar a revisao de conteudos vistos em aula;
- oferecer atividades interativas em HTML para uso rapido no navegador;
- manter um acervo organizado por materia.

## Disciplinas ministradas

Atualmente ministro aulas das seguintes disciplinas:

- `CAI` - Contabilidade Aplicada a Informatica
- `INF1` - Introducao a Informatica Basica
- `INF7` - Arquitetura de Software
- `INF8` - Backend com Python

## Estrutura do repositorio

Cada pasta representa uma disciplina. Dentro de cada pasta ficam os simulados, exercicios e paginas interativas relacionados ao conteudo da materia.

Estrutura atual:

```text
simulados/
├── CAI/
│   ├── cronicas-da-contabilidade.html
│   ├── simulado_cai_interativo.html
│   ├── simulado2_cai.html
│   └── simulado3_cai.html
└── README.md
```

1. Clone ou baixe este repositorio.
2. Acesse a pasta da disciplina desejada.
3. Abra o arquivo `.html` no navegador.
4. Utilize o simulado com os alunos ou como atividade de revisao.
