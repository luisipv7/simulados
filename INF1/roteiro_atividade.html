INF1 Introdução ao CLI do Windows Unidade 2 — Automação com .BAT
ATIVIDADE PRÁTICA ⏱ 90 MIN INDIVIDUAL

Meu Computador
Organizado _

Crie um script .bat que organize automaticamente arquivos em pastas por família, gere documentos de texto e abra o Paint para produzir uma imagem — tudo via linha de comando.

▸ Objetivo de aprendizagem

Ao final desta atividade você será capaz de criar arquivos .bat, usar variáveis de ambiente, criar e navegar entre pastas com o CMD, gerar arquivos de texto automaticamente, abrir programas nativos do Windows e organizar arquivos em uma estrutura de diretórios por família de tipo.

@echo off
@echo off
Impede que cada linha do script seja exibida no terminal. Coloque sempre na 1ª linha.
REM
REM isso é um comentário
Linha de comentário. O sistema ignora tudo que vem após REM.
echo
echo Olá mundo
echo texto > arquivo.txt
echo mais >> arquivo.txt
Exibe texto no terminal. Com > cria/sobrescreve um arquivo. Com >> adiciona ao fim.
mkdir
mkdir NomeDaPasta
mkdir "C:\Caminho\Pasta"
Cria uma ou mais pastas. Use aspas se o nome tiver espaços.
set
set NOME=valor
echo %NOME%
Cria uma variável. Use %VARIAVEL% para recuperar o valor depois.
copy / move
copy origem destino\
move arquivo "pasta\"
copy duplica o arquivo. move transfere para outro lugar.
start
start mspaint
start /wait mspaint
Abre um programa. /wait faz o script pausar até o programa ser fechado.
if exist
if exist "arquivo.txt" (
  echo existe
) else (
  echo não existe
)
Verifica se um arquivo ou pasta existe antes de agir.
pause
pause
pause >nul
Pausa o script esperando o usuário. >nul esconde a mensagem padrão.
cls / title
cls
title Meu Script
cls limpa a tela. title muda o título da janela do terminal.
tree
tree "C:\pasta" /f
Exibe a árvore de diretórios visualmente. /f inclui os arquivos.
explorer
explorer "C:\pasta"
Abre o Explorador de Arquivos do Windows na pasta indicada.

01
Abrir o Bloco de Notas e criar o arquivo .bat
Todo script .bat é um arquivo de texto simples. Abra o Bloco de Notas, escreva os comandos e salve com a extensão .bat — não .txt!
como abrir
Win+R  →  notepad   ← abre o Bloco de Notas
ao salvar (Ctrl+S)
Nome do arquivo:  projeto_meu_computador.bat
Tipo:             Todos os Arquivos (*.*)   ← IMPORTANTE!
💡 Se você salvar como .txt, o Windows não vai executar o script. Sempre escolha "Todos os Arquivos" ao salvar um .bat.
02
Configurar o cabeçalho e variáveis
Comece o script desligando o eco dos comandos, definindo um título e criando uma variável com o caminho da pasta principal do projeto.
código
@echo off
REM Desliga a exibição de cada linha no terminal

title Projeto INF1
REM Muda o título da janela do CMD

set PROJETO=%USERPROFILE%\Desktop\ProjetoINF1
REM %USERPROFILE% aponta para C:\Users\SeuNome automaticamente

echo Iniciando projeto em: %PROJETO%
pause
💡 Variáveis de ambiente como %USERPROFILE%, %DATE%, %TIME% e %USERNAME% já existem no Windows — você pode usar sem precisar criar.
03
Criar a estrutura de pastas por família
Use mkdir para criar a pasta principal e todas as subpastas. Lembre-se: cada família de arquivo tem sua própria pasta!
código
mkdir "%PROJETO%"
mkdir "%PROJETO%\Imagens"
mkdir "%PROJETO%\Documentos"
mkdir "%PROJETO%\Musicas"
mkdir "%PROJETO%\Videos"
mkdir "%PROJETO%\Programas"

echo Pastas criadas com sucesso!
⚠ Se a pasta já existir, o mkdir exibe um erro — mas o script continua rodando normalmente. Isso é esperado na segunda execução.
04
Gerar arquivos de texto automaticamente
Crie pelo menos dois arquivos .txt dentro da pasta Documentos usando echo com os operadores > e >>.
código
REM Cria o arquivo (> sobrescreve se já existir)
echo Meu primeiro documento via CMD! > "%PROJETO%\Documentos\bemvindo.txt"
echo Criado em: %DATE%                >> "%PROJETO%\Documentos\bemvindo.txt"
echo Usuario: %USERNAME%               >> "%PROJETO%\Documentos\bemvindo.txt"

REM Cria um segundo documento
echo Minhas materias favoritas:       >  "%PROJETO%\Documentos\materias.txt"
echo - Informatica                    >> "%PROJETO%\Documentos\materias.txt"
echo - Matematica                     >> "%PROJETO%\Documentos\materias.txt"
05
Abrir o Paint e salvar o desenho na pasta Imagens
Use start /wait mspaint para abrir o Paint. O /wait faz o script esperar você terminar de desenhar. Quando fechar o Paint, o script continua!
código
echo O Paint vai abrir. Faca um desenho e salve em:
echo %PROJETO%\Imagens\meu_desenho.bmp
echo Depois FECHE o Paint para continuar.
pause

start /wait mspaint
REM O script fica pausado aqui até você fechar o Paint!

echo Paint fechado. Verificando...

REM Verifica se o arquivo foi salvo corretamente
if exist "%PROJETO%\Imagens\meu_desenho.bmp" (
    echo [OK] Desenho encontrado!
) else (
    echo [AVISO] Salve o desenho na pasta Imagens!
)
💡 No Paint: Arquivo → Salvar como → BMP e navegue até a pasta Imagens do seu projeto.
06
Copiar o próprio script para a pasta Programas
Um script que se auto-organiza! Copie o arquivo .bat para a pasta Programas usando a variável especial %~f0 — ela contém o caminho do próprio script.
código
copy "%~f0" "%PROJETO%\Programas\meu_projeto.bat" >nul
echo Script copiado para Programas!

REM %~f0 = caminho completo do arquivo .bat em execução
07
Exibir a estrutura final e abrir o Explorador
Finalize o script mostrando a árvore de pastas e arquivos com tree, depois abra o Explorador para confirmar visualmente o resultado.
código
echo ====================================
echo  ESTRUTURA FINAL DO PROJETO:
echo ====================================
tree "%PROJETO%" /f

explorer "%PROJETO%"
REM Abre o Explorador de Arquivos na pasta do projeto

echo Projeto concluido!
pause >nul

▸ Desafio extra

Faça o seu script ir além!

  • Crie pelo menos 3 documentos .txt com conteúdos diferentes — um diário, uma lista, uma receita, o que quiser.
  • Adicione uma subpasta dentro de Imagens chamada Rascunhos e salve um segundo desenho lá.
  • Crie um arquivo de log em LEIA-ME.txt com seu nome, data, hora e nome do computador usando variáveis de ambiente.
  • Use if exist para verificar cada arquivo criado e exibir [OK] ou [ERRO] para cada um.
  • Crie um tema diferente — em vez de escola, organize como se fosse um projeto de jogo (pasta Sprites, Sons, Mapas, Scripts).
